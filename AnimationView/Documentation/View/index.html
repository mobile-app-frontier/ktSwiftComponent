shimmering(active:​duration:​bounce:​delay:​) 

@ViewBuilder func shimmering(
        active: Bool = true, duration: Double = 1.5, bounce: Bool = false, delay: Double = 0
    ) -> some View

Adds an animated shimmering effect to any view, typically to show that an operation is in progress.

Parameters

Name Type Description
active Bool

Convenience parameter to conditionally enable the effect. Defaults to true.
duration Double

The duration of a shimmer cycle in seconds. Default: 1.5.
bounce Bool

Whether to bounce (reverse) the animation back and forth. Defaults to false.
delay Double

A delay in seconds. Defaults to 0.

shimmering(active:​animation:​) 

@ViewBuilder func shimmering(active: Bool = true, animation: Animation = ShimmerModifier.defaultAnimation) -> some View

Adds an animated shimmering effect to any view, typically to show that an operation is in progress.

Parameters

Name Type Description
active Bool

Convenience parameter to conditionally enable the effect. Defaults to true.
animation Animation

A custom animation. The default animation is .linear(duration: 1.5).repeatForever(autoreverses: false).
